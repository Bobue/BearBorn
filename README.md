# 3TileM

모바일 환경을 목표로 개발 중인 턴 기반 게임 프로젝트입니다.  
타일/카드 시스템을 활용한 전투 구조를 가진 팀 프로젝트입니다.

---

## 🎮 프로젝트 개요
- **플랫폼**: Mobile
- **장르**: 턴 기반 / 전략 / 퍼즐
- **개발 형태**: 팀 프로젝트
- **개발 인원**
  - 개발자 3명
  - 기획자 3명
  - 디자이너 2명
- **개발 기간**: 2025.03 ~ 2026.01  
  (본인 참여 기간: 2025.09 ~ 2026.01)

---

## 🛠 사용 기술
- **Engine**: Unity 2022.3.62f3
- **Language**: C#
- **Library**
  - DOTween (UI 및 이펙트 연출)
- **Version Control**: Git / GitHub

---

## 👤 담당 역할
- 플레이어 및 적 스킬 로직 구현
- 스킬 발동 조건 및 효과 처리
- 전투 중 이펙트 연출 및 타이밍 제어
- DOTween을 활용한 이펙트 및 UI 연출 구현
- 기존 전투 흐름과 구조에 맞춰 기능 단위 스크립트 구현
- 기획 변경 사항에 따른 스킬 효과 수정 및 대응

---

## 🧩 핵심 구현 내용

### ⚔️ 스킬 시스템
- 스킬별 발동 조건에 따른 로직 처리
- 타겟 지정, 데미지 계산, 상태 효과 적용 구현
- 전투 흐름을 해치지 않도록 스킬 처리 순서 고려

---

### ✨ 이펙트 및 연출 처리
- 스킬 사용 시 시각적 피드백 강화를 위한 이펙트 적용
- DOTween을 활용한 타이밍 기반 연출 구현
- 로직 처리와 연출이 자연스럽게 이어지도록 조정

---

### 🧠 협업 관점에서의 작업 방식
- 이미 정의된 턴 진행 구조를 이해하고 그에 맞춰 기능 구현
- 기획자와 협업하여 스킬 효과 및 수치 조정
- 디자이너와 협업하여 이펙트 타이밍 및 표현 개선
- 팀 내 코드 흐름을 해치지 않도록 기존 구조 존중

---


## ✍ 개발자 노트
- 본 프로젝트에서 저는 **전투 플레이의 체감과 완성도를 높이는 구현**을 주로 담당했습니다.
- 스킬 로직과 이펙트 연출이 자연스럽게 연결되도록 타이밍 조정에 많은 신경을 썼습니다.
- 팀 프로젝트 환경에서 기존 구조를 이해하고, 그 안에서 안정적으로 기능을 추가하는 경험을 쌓았습니다.
